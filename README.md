# Netflix ETL Pipeline

Simple ETL pipeline built with Python and PostgreSQL.

The project extracts Netflix dataset, performs basic transformations, and loads the data into a PostgreSQL database running in Docker.

## Project architecture

CSV Dataset (Kaggle)
        ↓
Extract (pandas)
        ↓
Transform (data cleaning)
        ↓
Load (SQLAlchemy)
        ↓
PostgreSQL (Docker)

## Tech stack

- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- Docker
